Preguntas Frecuentes

¿Cuánto tiempo de anticipación necesito para reservar?

Dependiendo de mi disponibilidad, puedes reservar desde 2 semanas hasta 3 meses de anticipación. Para consultas, puedo agendar dentro de una semana.

¿Trabajas solo con citas o también walk-ins?

Trabajo exclusivamente con citas programadas para asegurar la calidad y atención personalizada que cada cliente merece.

¿Cuál es el precio promedio de un tatuaje?

Los precios varían según tamaño, complejidad y tiempo. Para una consulta personalizada, contáctame directamente para una cotización precisa.

¿Qué formas de pago aceptas?

Acepto efectivo, tarjetas de débito y crédito, y transferencias bancarias. El depósito para reservar se puede hacer por cualquiera de estos métodos.

¿Ofreces diseños personalizados?

Sí, todos mis trabajos son 100% personalizados. Cada diseño es único y creado específicamente para cada cliente basado en sus preferencias y significado.

¿Qué cuidados necesito después de tatuarme?

Te proporcionaré instrucciones detalladas de cuidado después de cada sesión. Generalmente, incluye limpieza suave, hidratación y protección solar.
